# Getting Started

```bash 
 # cloning the repo
 git clone https://github.com/jayeshsojitra103/grid-ui.git

 cd grid-ui
```

```bash
# Environment Variables
To run this project, you need to set up environment variables for API access. Create a .env file in the root of your project directory and add the following lines: 

Replace <your-api-url> with the URL of your Geo DB API and <your-api-key> with your API key.
```

```bash 
# install dependencies
 npm i
 npm start

 Runs the app in the development mode.
 Open http://localhost:3000 to view it in your browser.

```

```bash
# Features
- Table: Displays data in a table format with support for loading and error states.

- SearchBox: Allows users to perform searches and filter data.

- Limit Selector: Lets users adjust the number of items displayed per page.

- Pagination Component: Provides navigation for paginated data with next and previous page buttons.

```
